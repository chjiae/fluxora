# 用量趋势聚合

请求日志、概要和趋势接口均在 Platform，先按权限收缩范围：平台管理员必须明确传入 `tenantId`；租户管理员仅能查看本租户；普通成员仅能查看本人。`requestId` 详情也执行同一约束，不能因全局唯一绕过隔离。

业务时区由 `OBSERVABILITY_BUSINESS_TIME_ZONE`（默认 `Asia/Shanghai`）统一控制。今天为当天 00:00 至当前时刻并按小时聚合；近 7 天和近 30 天均包含今天并按自然日聚合。每个桶对 Token 返回完整、部分或未知状态；全未知返回 `null`，不会伪装成零。理论金额仅聚合 `CALCULATED` 记录，页面明确提示未知或部分数据不在统计内。
